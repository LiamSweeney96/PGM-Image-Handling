In order to compile the files in this implementation run one of the following commands:

1. make all
2. make pgmEcho
3. make pgmComp
4. make pgma2b
5. make pgmb2a
6. make pgmReduce

Upon doing so, the files will compile. Once this is done, run one of the following commands to run the relevant program:

1. ./pgmEcho inputImage.pgm outputImage.pgm
2. ./pgmComp inputImage.pgm inputImage.pgm
3. ./pgma2b inputImage.pgm outputImage.pgm
4. ./pgmb2a inputImage.pgm outputImage.pgm
5. ./pgmReduce inputImage.pgm reduction_factor outputImage.pgm

Once this is done, the selected program will run.
